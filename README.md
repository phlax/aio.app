aio.app

Build status
------------

Installation
------------

Install with:

``

  pip install aio.app

``

Configuration
-------------

By default the aio command will look for the following configuration files

   - aio.conf
   
   - etc/aio.conf
   
   - /etc/aio.conf

Once it has found a config file it uses that one
